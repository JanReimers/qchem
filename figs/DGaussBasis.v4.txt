#  calculations. To fit the charge density, the A1 basis set is recommended
#  for use with the DZVP orbital basis set, while the A2 set is recommended for
#  use with the DZVP2 or TZVP orbital basis sets. For references, see:

#  (1) N. Godbout, D.R. Salahub, J. Andzelm and E. Wimmer, Can. J. Chem. 70,
#      560 (1992);
#  (2) C. Sosa, J. Andzelm, B.C. Elkin, E. Wimmer, K.D. Dobbs and D.A. Dixon,

# DGauss 4.0 basis set format:
# ---------------------------
#
#  For DGauss 4.0, the following data format is used to list basis sets:
#
#
#
#  atom basis [format]
#
#  Nshells
#
#  l  Np  [Nc]  [scale=factor]
#
#  a column       one or more
#     of          columns of
#  primitive      contraction 
#  exponents      coefficients
#
#
#
#  All fields in square brackets are optional.
#
#  All data is listed in free format and read in a case-insensitive manner.
#
#  The first line lists atom, basis and format labels. The atom label must be
#  an atomic symbol from the periodic table. The format label must be either
#  "DGauss3.0" or "DGauss4.0". The default format is "DGauss4.0". However, any
#  basis set previously formatted for DGauss 3.0 can be read by adding it to
#  this file and designating its format to be "DGauss3.0".
#
#  The second line specifies the number of shells of contracted Gaussian-type
#  functions (GTF) in the basis set.
#
#  The third line specifies the angular momentum, l, and the number of primitive
#  GTF, Np, in the first shell. The angular momentum can be specified with
#  either integers or the conventional symbols:
#
#     l = 0, 1, 2, 3, 4, 5, 6, etc.
#         S, P, D, F, G, H, I
#
#  In addition, the symbols "SP" and "SPD" can be used to specify shells that
#  share primitive exponents between functions of s and p angular momentum
#  and between functions of s, p and d angular momentum, respectively.
#
#  In the case of general contractions, this line also specifies the number of
#  contractions, Nc, that will share the same set of primitive.
#
#  This line can also specify a scale factor for the primitive exponents. If a
#  factor is listed, all of the exponents will be scaled by the square of this
#  factor (i.e., exponent = exponent * factor**2).
#  
#  Following the third line, the primitive exponents and contraction
#  coefficients of the first shell are listed in columns. The exponents are
#  listed in the first column followed by the coefficients. The number of
#  columns of coefficients varies with the type of shell. A single column of
#  coefficients should be listed for segmented contractions, while Nc columns
#  should be listed for general contractions. For "SP" or "SPD" shells, the
#  coefficients for the s-functions should be listed in the second column,
#  followed by those for the p-functions, etc. If the shell is comprised of
#  just one primitive, then no coefficients need be listed.
#
#  Returning to the third line, this format is repeated for each subsequent
#  shell of the basis set.
#
#  Comments lines begin with the symbol "#" and can placed anywhere but
#  immediately before or between rows of exponents and coefficients.
#
#  Blank lines are simply skipped.
#
